# RoomBookingService

## Задание: API для управления бронированием помещений (Room Booking System)

### Описание:
Необходимо разработать RESTful API для системы бронирования помещений в офисе. Система должна позволять пользователям бронировать переговорные комнаты на определенные временные слоты, управлять бронированиями, а также проверять доступность комнат.

### Функциональные требования:

Опции комнаты: 
Создание, получение, удаление опций комнаты(проектор, телевизор, доска и т.д.)

Помещения:
Создание, получение, редактирование, удаление комнаты.
Получение списка всех комнат с возможностью фильтрации по вместимости и наличию оборудования (проектор, телевизор, доска и т.д.).

Бронирования:
Создание, получение, редактирование, удаление бронирования.
Возможность просмотра всех бронирований по комнате с пейджингом.

Пользователи:
Создание, получение, редактирование, удаление пользователя.
Получение списка всех пользователей с пейджингом.


### Уведомления (бонус):

Отправка уведомлений (email) участникам о создании или изменении бронирования.
